Principais termos da TI

Termo Padrão Definição
Hardware Parte física do computador, como processador, memória RAM e HD.
Software Programas que rodam no computador, incluindo sistemas operacionais e aplicativos.
Rede LAN Rede local que conecta computadores e dispositivos em um mesmo espaço físico.
SSD Unidade de armazenamento mais rápida que HD, melhora a performance do sistema.
VPN Rede privada virtual que protege a conexão e garante segurança na troca de dados.
Firewall Sistema que controla e protege o tráfego de dados em uma rede, prevenindo acessos não autorizados.
Cloud Computing Computação em nuvem; permite armazenar e processar dados em servidores remotos acessíveis via internet.
IoT Internet das Coisas; dispositivos conectados à internet que trocam informações entre si.
Big Data Conjunto de dados muito grandes ou complexos que exigem ferramentas especiais para análise e processamento.
API Interface de Programação de Aplicações; permite que softwares diferentes se comuniquem entre si.
HTML Linguagem de marcação usada para estruturar páginas web.
CSS Linguagem de estilo que define a aparência de páginas web, como cores, fontes e layout.
JavaScript Linguagem de programação usada principalmente para tornar páginas web interativas.
Malware Software malicioso criado para prejudicar ou explorar computadores e redes.
Phishing Técnica de fraude online que tenta enganar usuários para obter dados pessoais ou financeiros.
Backup Cópia de segurança de dados, importante para evitar perda de informações importantes.
